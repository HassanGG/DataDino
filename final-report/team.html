Reflections

What have we learnt

  • React - we learned how to use it (to a basic degree)
  • We got great experience working in two groups in parallel without much difficulty. We did this by writing API contracts that the backend developed towards and the frontend expected. It worked very well and we were able to replace the mock server with requests to the actual backend very quickly.
  • We got more practice pair programming - this was a very enjoyable and productive use of time.

What would we use differently

  • Done more research into Spring boot; there is a lot of magic - things happening automatically that are difficult to debug and having more knowledge about Spring internals may have helped.
  • Same idea with React; knowing more about component lifecycles would have been of great use debugging issues regarding reactivity.
  • We barely did any code reviews; our main branch could have been cleaner at times.
